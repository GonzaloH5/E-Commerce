# TechShop - E-commerce

Este es el repositorio de **TechShop**, un proyecto de e-commerce desarrollado para ofrecer la mejor experiencia de compra en el rubro de la informática.

## Acerca del Proyecto
TechShop es una tienda virtual muy intuitiva, orientada exclusivamente a mostrar y vender productos tecnológicos. 

## ¿Qué opciones vendemos?
Nuestro catálogo está pensado para armar o mejorar tu setup de PC, destacando:
- Componentes de hardware principales (tarjetas de video, procesadores, motherboards y memoria RAM).
- Periféricos gamer, de oficina y de diseño (teclados mecánicos, mouses de precisión, auriculares, micrófonos).
- Monitores y todo tipo de accesorios complementarios.
